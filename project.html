Overview

소규모 기업을 대상으로 한 웹 기반 ERP 서비스 'EsRP'. 5인 팀 프로젝트로 약 1개월에 걸쳐 완성한 웹 애플리케이션입니다. 프로그램을 설치하지 않고 회원 인증만으로 웹에서 ERP 시스템을 이용할 수 있습니다. 간단한 UI 구성과 조작을 기반으로 하여 핵심 기능 위주로 구현하였습니다.

Teamwork

프론트엔드부터 백엔드까지 팀원들과 분업하여 프로그래밍하였습니다. MVC 패턴을 사용한 JAVA·JSP 프로그래밍에 참여하였으며, 전반적인 디자인 컨셉 기획부터 UI/UX 디자인, CSS, PPT 제작은 본인이 전담하였습니다. 프론트단과 백단을 오고 가며 프로그래밍 기술을 발전시키고, 팀원들과 적극적으로 소통하며 목적을 달성한 프로젝트였습니다.

Develop

Oracle, Java, Spring MVC pattern, myBatis, Apache Tomcat

Function

[1] 회원 인증
[2] 인사·영업·자재 관리
[3] 입출력 자동화 기능
